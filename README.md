## Installation
 [+] Install git & php
```
apt-get install git -y
apt-get install php -y
```
 [+] Clone this Repo
```
git clone https://github.com/sanjidtk/IG-aakri.git

cd IG-aakri
```
 [+] Inside Repo Unzip 'vendor.zip'
```
unzip vendor.zip
```
#### Run the Login
```
php login.php
```
#### Run the Tool
```
php igscraper.php
```
